# Network Port Scanner (Python)

This project is a multithreaded TCP port scanner built using Python.

## Features

* Scans custom port ranges
* Multithreading for fast execution
* Banner grabbing to identify services

## Tech Used

* Python
* Socket Programming
* Threading

## How to Run

1. Run: python port_scanner.py
2. Enter target and port range

## Learning Outcome

* Understood TCP connections and port scanning
* Explored basic network security concepts
